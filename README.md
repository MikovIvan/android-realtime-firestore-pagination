# Realtime Firestore Pagination Sample App

This project is a sample app for realtime firestore pagination on Android

## Libraries and Tools Used

- Kotlin
- Android Architecture Components
    - Lifecycle and ViewModel
    - Paging
- Kotlin Coroutines
- Dagger 2
